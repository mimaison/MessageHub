# Message Hub에서 Kafka API를 사용하는 데 필요한 사항

* [API용 Javadoc ![외부 링크 아이콘](../../icons/launch-glyph.svg "외부 링크 아이콘")](http://kafka.apache.org/0102/javadoc/index.html){:new_window}
